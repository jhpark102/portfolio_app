# 개인 포트폴리오 1페이지 웹사이트

## 기능 명세서 (Functional Specification)

---

## 1. 프로젝트 개요

### 1.1 목적

* 개인 개발자의 **경력·전문성·신뢰성**을 한눈에 전달
* 이력서 대체 또는 보완 용도
* 채용 담당자 / 협업 파트너가 **5분 이내 핵심 파악 가능**

### 1.2 기본 조건

* **Single Page (One Page Scroll)**
* 데스크톱 / 모바일 **반응형**
* SEO 기본 대응
* 퍼포먼스 최적화 고려

---

## 2. 전체 페이지 구조

```
[ Header / Navigation ]
[ Hero (Intro) ]
[ About Me ]
[ Skills / Tech Stack ]
[ Experience / Projects ]
[ Portfolio ]
[ Contact ]
[ Footer ]
```

---

## 3. Header / Navigation

### 기능 요구사항

* 고정 헤더 (스크롤 시 상단 고정)
* 메뉴 클릭 시 **해당 섹션으로 부드러운 스크롤 이동**
* 모바일에서는 햄버거 메뉴로 전환

### 메뉴 항목

* Home
* About
* Skills
* Experience
* Portfolio
* Contact

---

## 4. Hero 영역 (첫 화면)

### 기능 요구사항

* 페이지 로딩 시 가장 먼저 노출
* 개발자 이름 + 한 줄 전문 요약 문구
* CTA 버튼 1~2개

  * 예: `포트폴리오 보기`, `연락하기`

### 기술 요소

* CSS 애니메이션 또는 간단한 인터랙션
* 이미지 사용 시 lazy loading

---

## 5. About Me

### 기능 요구사항

* 개발자 소개 텍스트
* 총 경력 연차, 주요 분야 요약
* 간단한 프로필 이미지 (선택)

### 구현 포인트

* 텍스트 중심, 과도한 장식 배제
* 모바일 가독성 우선

---

## 6. Skills / Tech Stack

### 기능 요구사항

* 사용 기술 목록 시각화

  * 언어 / 프레임워크 / 인프라 / 툴 등 분류
* 숙련도 표현 (아이콘, 바, 태그 등)

### 기술 요소

* 정적 데이터 기반 렌더링
* 반복 구조 컴포넌트화

---

## 7. Experience / Projects (경력 요약)

### 기능 요구사항

* 주요 프로젝트 또는 경력 타임라인 형태
* 각 항목에 포함될 정보:

  * 프로젝트명
  * 기간
  * 역할
  * 사용 기술
  * 핵심 성과

### 구현 포인트

* 리스트 기반 구성
* 모바일에서는 카드 형태로 재배치

---

## 8. Portfolio (상세 작업물)

### 기능 요구사항

* 대표 프로젝트 카드 리스트
* 카드 클릭 시:

  * 외부 링크 이동 또는
  * 간단한 상세 모달 표시

### 카드 구성 요소

* 프로젝트 썸네일
* 프로젝트명
* 간단 설명
* 사용 기술 태그

---

## 9. Contact

### 기능 요구사항

* 연락 수단 제공

  * 이메일
  * GitHub
  * LinkedIn (선택)
* 간단한 Contact Form (선택)

### Contact Form (선택 구현)

* 입력 항목: 이름 / 이메일 / 메시지
* 필수 입력값 검증
* 전송 후 성공 메시지 표시

---

## 10. Footer

### 기능 요구사항

* 저작권 문구
* 간단한 링크
* 연도 자동 갱신

---

## 11. 공통 기능 및 비기능 요구사항

### 반응형

* Mobile / Tablet / Desktop 대응
* Breakpoint 기준 명확히 정의

### 접근성

* 시맨틱 태그 사용
* 키보드 네비게이션 기본 대응

### 성능

* 불필요한 JS 최소화
* 이미지 최적화
* 초기 로딩 속도 고려

### SEO

* 메타 태그 구성
* Open Graph 기본 적용

---

## 12. 기술 스택 (예시, 변경 가능)

* HTML5 / CSS3
* JavaScript (또는 TypeScript)
* React / Next.js (선택)
* Tailwind CSS (선택)
